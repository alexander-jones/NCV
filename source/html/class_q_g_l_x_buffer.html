Detailed Description

An QGL extension class for managing OpenGL buffers. Extends functionality of QGLBuffer to include Texture Buffers.

Author
Alex Jones

Member Enumeration Documentation

enum QGLXBuffer::Type

This enum defines the type of GL buffer object to create with QGLXBuffer.

Enumerator:
VertexBuffer 

Vertex buffer object for use when specifing vertex arrays.

IndexBuffer 

Index buffer object for use with glDrawElements / glDrawElementsInstanced.

PixelPackBuffer 

Pixel pack buffer object for reading pixel data from the GL server (for example, with glReadPixels()). Not supported under OpenGL/ES.

PixelUnpackBuffer 

Pixel unpack buffer object for writing pixel data to the GL server (for example, with glTexImage2D()). Not supported under OpenGL/ES.

TextureBuffer 

Texture buffer object used for specifing buffer data as a texture.
